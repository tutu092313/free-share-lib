📋一、项目概述

"随心资料分享库"是一款面向高校学生的通用型学习资料分享 Web 小程序， 旨在解决学生群体资料分散、难以统一检索与分享的痛点。 项目采用纯前端实现，无需后端与数据库，所有数据持久化于浏览器 localStorage， 双击 index.html 即可完整体验全部功能。

🎯 核心目标

通用性（任意专业/领域）、零门槛（无需后端）、多格式（PDF/Word/图片）、可扩展（对接真实后端）

📦 项目规模

2 个源文件（index.html 419 行 + app.js 1183 行）；16 份内置示例资料；6 个单页视图

功能清单

  • 用户模块：注册 / 登录 / 登出 / 个人中心 / 体验账号一键登录
  • 资料上传：PDF / Word(.docx) / 图片，≤8MB，拖拽 + 点击，自动生成缩略图
  • 专业 × 领域双维度分类（20 个专业 + 15 个领域，用户可自由扩展）
  • 全文搜索（标题、描述、标签、专业名）+ 三种排序（最新/最热/下载最多）
  • 多格式在线预览（PDF.js / mammoth.js / 原生图片）
  • 评论 / 点赞 / 收藏 / 下载 / 删除自己上传的资料
  • 投喂系统：JSON 批量入库 / 填表投喂 / 文件批量拖入
  • 响应式适配：桌面4栏 + 移动底部 Tab 栏

7.6 缩略图生成

三种情况：图片直接缩放到 320×220 Canvas；PDF 用 PDF.js 渲染第一页；Word 和其他类型用彩色文字图标代替。

javascriptapp.js · 第 275–332 行
async function makeThumbnail(file) {
  const type = (file.type || '').toLowerCase();
  const name = file.name.toLowerCase();

  // 1. 图片 → Canvas 缩放
  if (type.startsWith('image/')) {
    return await new Promise(resolve => {
      const img = new Image();
      img.onload = () => {
        const c = document.createElement('canvas');
        c.width = 320; c.height = 220;
        const ratio = Math.min(320 / img.width, 220 / img.height);
        c.getContext('2d').drawImage(img, (320 - img.width*ratio)/2, ...);
        resolve(c.toDataURL('image/jpeg', 0.8));
      };
    });
  }

  // 2. PDF → PDF.js 渲染第一页
  if (name.endsWith('.pdf')) {
    const buf = await file.arrayBuffer();
    const pdf = await pdfjsLib.getDocument({ data: buf }).promise;
    const page = await pdf.getPage(1);
    const canvas = document.createElement('canvas');
    canvas.width = 320; canvas.height = 220;
    await page.render({ canvasContext: canvas.getContext('2d'), viewport: ... }).promise;
    return canvas.toDataURL('image/jpeg', 0.8);
  }

  // 3. Word / 其他 → 彩色文字图标
  return iconThumb(name.endsWith('.docx') ? 'docx' : 'file');
}

// 生成纯色文字缩略图（PDF=红 / DOCX=蓝 / IMG=绿 / FILE=灰）
function iconThumb(kind) {
  const color = { pdf: '#dc2626', docx: '#2563eb', image: '#059669', file: '#64748b' }[kind];
  const c = document.createElement('canvas');
  c.width = 320; c.height = 220;
  const ctx = c.getContext('2d');
  ctx.fillStyle = '#f8fafc'; ctx.fillRect(0, 0, 320, 220);
  ctx.fillStyle = color; ctx.font = 'bold 56px sans-serif';
  ctx.fillText(label, 160, 110);
  return c.toDataURL('image/jpeg', 0.8);
}
